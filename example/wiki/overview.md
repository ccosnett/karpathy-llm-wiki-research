# Overview

This wiki is a tiny, strict example of Karpathy's pattern applied to Karpathy's own idea.

## Scope

The wiki answers one narrow question:

What is an LLM wiki, according to Karpathy's April 2026 framing?

## Current Shape

- one entity page
- one concept page
- two source-summary pages
- one comparison page

## Key Takeaway

The central move is to place a maintained markdown wiki between the user and the raw corpus, so synthesis compounds instead of being recomputed from scratch each time.
